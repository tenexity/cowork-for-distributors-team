## How this file is used

Claude reads this when:
- You ask open-ended questions ("what should I focus on?")
- You start a session without specifying a task
- The System Review runs (monthly) and looks for whether your priorities are getting actual attention

The Onboarding Coach captures the initial set. The System Review proposes updates over time as your patterns emerge from session memory. You always approve changes — the system never silently rewrites your priorities.
