# Me

This is the **individual user layer** of CoWork for Distributors — the file that describes *you*. Each employee has their own copy.

The Setup User skill populates this in about 5 minutes. It reads the company-level files in `ABOUT ME/COMPANY/` (which describe the business itself) so it doesn't waste time re-asking about the ERP, branches, or vendor list. It only asks about *you*.

Update this file when your role, branch, or focus changes.
