# Tools and context — company stack

Maps the tools the company uses and where information lives. Every employee in this workspace inherits this stack.

The GM or company admin populates this once via Setup Company. Updates happen when a new tool is added (a new portal, a new buying-group system, a new ERP module).

Individual employees do not edit this file.

## How this file is used

Claude checks this map when a task involves looking up information, finding documents, or understanding how the company's tools connect. Tools with direct CoWork connectors get accessed automatically. Tools without connectors (the ERP, manufacturer portals) get referenced by name so Claude knows what the user is asking about when they paste an export.
