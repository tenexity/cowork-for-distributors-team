## How to use this artifact

Fill it in Monday morning before the leadership call (15 minutes). Talk through it during the call. Save the version each week — over time you'll have a story of how the branch is trending.

If a metric isn't relevant for your branch (say you don't track average ticket), delete the row. If something you DO track isn't here (you have a specific lead-time KPI for HVAC equipment, say), add a row. The template is yours to customise.
